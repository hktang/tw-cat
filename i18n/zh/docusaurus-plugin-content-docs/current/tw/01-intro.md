# 1. 写在前面

我们将通过本课程了解如何借助技术的力量提高效率。但首先必须强调，
文字素养和创造力才是文字工作者的立身之本。本课程虽然不直接涉及这些内容，
但请各位同学切记：善用工具，而不是成为它的奴隶。
