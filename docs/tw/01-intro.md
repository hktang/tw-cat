# 1. 写在前面

我们将通过本课程了解如何借助技术的力量提高技术写作的效率。
但首先必须强调，文字素养和创造力才是文字工作者的立身之本。
这虽然并不是本课程的重点，但请同学们切记：
善用工具，而不是成为工具的奴隶。
