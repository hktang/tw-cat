# 7.4 DITA 的优缺点

DITA 作为技术写作的工业标准，有开源、便于版本控制、专业软件支持、便于索引等诸多优势。
但随着信息技术的发展，DITA的一些劣势也纷纷显露。[[1]][1]

- 不兼容现有Web框架和JavaScript技术
- 简化的信息类型尚存争议
- XML编写相对困难
- 不兼容非DITA内容
- 不易兼容现代搜索技术
